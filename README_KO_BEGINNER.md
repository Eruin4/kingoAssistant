# kingoAssistant 쉬운 설명서

kingoAssistant는 안드로이드에서 음성으로 일정과 할 일을 관리하는 프로젝트입니다.
사용자가 앱에 말을 하면, 서버가 음성을 글자로 바꾸고 AI가 내용을 해석해서 일정이나 할 일로 정리합니다.

예를 들어 사용자는 다음처럼 말할 수 있습니다.

```text
내일 오후 3시에 회의 추가해줘
선형대수 문제풀이 할 일로 추가해줘
다음 주 월요일 과제 일정 추가해줘
```

앱은 AI가 만든 결과를 바로 저장하지 않고 먼저 제안으로 보여줍니다.
사용자가 제안을 수락하면 실제 일정이나 할 일 목록에 반영됩니다.

## 전체 구조

프로젝트는 크게 안드로이드 앱과 서버로 나뉩니다.

```text
안드로이드 앱 -> HomeVoice 서버 -> Whisper / KingoGPT
```

각 부분의 역할은 다음과 같습니다.

- 안드로이드 앱: 사용자가 보는 화면입니다. 음성을 녹음하고 일정, 할 일, 제안 목록을 보여줍니다.
- HomeVoice 서버: 앱에서 받은 요청을 처리합니다. 음성 파일을 받고, 일정 데이터를 저장합니다.
- Whisper: 음성 파일을 텍스트로 바꾸는 음성 인식 프로그램입니다.
- KingoGPT: 텍스트 내용을 읽고 일정 추가, 할 일 추가, 삭제, 조회 같은 명령으로 바꿉니다.

## 동작 흐름

1. 사용자가 안드로이드 앱에서 마이크 버튼을 누릅니다.
2. 앱이 사용자의 말을 WAV 음성 파일로 녹음합니다.
3. 앱이 음성 파일을 서버로 보냅니다.
4. 서버가 Whisper를 사용해서 음성을 텍스트로 변환합니다.
5. 서버가 변환된 텍스트와 현재 일정 정보를 KingoGPT에 보냅니다.
6. KingoGPT가 사용자의 의도를 구조화된 명령으로 바꿉니다.
7. 서버가 그 명령을 제안 목록에 저장합니다.
8. 앱이 제안을 화면에 보여줍니다.
9. 사용자가 수락하면 일정이나 할 일에 반영됩니다.

## 안드로이드 앱

안드로이드 앱에는 네 가지 주요 화면이 있습니다.

- Chat: 사용자의 명령과 AI 응답을 대화처럼 보여줍니다.
- Calendar: 일정 목록을 보여줍니다.
- Tasks: 할 일 목록을 보여줍니다.
- Settings: 서버 주소, API 키, 알림 시간, TTS 설정을 바꿀 수 있습니다.

앱에서 사용할 수 있는 기능은 다음과 같습니다.

- 마이크 버튼으로 음성 입력
- 텍스트로 명령 입력
- 홈 화면 위젯에서 음성 입력
- 일정과 할 일 확인
- AI가 만든 제안 수락 또는 거절
- 일정 또는 할 일을 왼쪽으로 밀어서 삭제
- 일정 알림
- AI 답변 음성 재생 켜기 또는 끄기

## 서버

서버는 안드로이드 앱에서 온 요청을 처리합니다.

서버가 하는 일은 다음과 같습니다.

- 앱에서 보낸 음성 파일 받기
- 음성을 텍스트로 변환하기
- KingoGPT에 텍스트 해석 요청하기
- 일정과 할 일 데이터 저장하기
- 제안 목록 관리하기
- 앱에 현재 일정 상태를 보내기

현재 운영 서버 주소는 다음과 같습니다.

```text
https://eruin.mooo.com/voice
```

## 저장되는 데이터

서버는 일정 데이터를 JSON 파일로 저장합니다.

주요 데이터는 다음과 같습니다.

- calendar_1: 첫 번째 일정표
- calendar_2: 두 번째 일정표
- tasks: 할 일 목록
- proposals: 아직 사용자가 수락하지 않은 제안 목록
- chat_history: 최근 대화 기록

## API

API는 앱과 서버가 서로 통신하기 위한 주소입니다.

주요 API는 다음과 같습니다.

```text
GET    /health              서버 상태 확인
GET    /schedule            현재 일정 가져오기
POST   /command             텍스트 명령 보내기
POST   /stt-command         음성 파일 보내기
POST   /proposal/accept     제안 수락하기
POST   /proposal/reject     제안 거절하기
POST   /proposal/accept-all 모든 제안 수락하기
DELETE /event/{id}          일정 삭제하기
DELETE /task/{id}           할 일 삭제하기
```

## KingoGPT 솔버

이 프로젝트는 KingoGPT 솔버를 직접 포함하지 않습니다.
운영 서버에서는 별도로 설치된 KingoGPT 프로젝트를 연결해서 사용합니다.

운영 서버의 실제 KingoGPT 솔버 경로는 다음과 같습니다.

```text
/home/eruin/kingoGPT/kingogpt_api_solver.py
```

Docker 컨테이너 안에서는 다음 경로로 연결됩니다.

```text
/app/kingoGPT/kingogpt_api_solver.py
```
